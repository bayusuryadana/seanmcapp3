# Seanmcapp3
Port application from [seanmcapp](https://github.com/bayusuryadana/seanmcapp)

## Feature
- Api: telegram webhook (bot)
- Scheduler: news, stock
- Web: wallet + stock
- Instagram fetcher

## Setup
1. Install Go
2. Install Node + Yarn
3. run backend `go run .`
4. run frontend `cd ui && yarn dev-local`
